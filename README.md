🧠 Samatva AI – Inner Clarity Engine

Samatva AI is an AI-powered decision-support system that helps users gain inner clarity before making decisions.
It combines NLP, AI reasoning, and the Sattva–Rajas–Tamas framework to guide users toward calm, balanced, and ethical choices.

🚀 Features
🧾 User Input Analysis (text/voice)
🤖 AI Chatbot for Decision Guidance
🧠 Emotion Detection (NLP-based)
⚖️ Guna Classification (Sattva, Rajas, Tamas)
🧘 Micro-Interventions (breathing, reflection prompts)
📊 Dashboard Visualization (Guna chart + stability index)
🏗️ Tech Stack

Frontend:

React.js
Tailwind CSS

Backend:

Python
FastAPI

AI / ML:

Hugging Face Transformers (Emotion Detection)
LLM API (Decision Reasoning)
Whisper (Speech-to-Text, optional)

Visualization:

Chart.js / Recharts

Database:

SQLite / Firebase
⚙️ How It Works
User enters a dilemma or emotional state
NLP model detects emotions (stress, confusion, etc.)
Guna Engine maps state → Sattva, Rajas, Tamas
AI chatbot provides clarity-based guidance
System suggests micro-interventions
Dashboard displays insights and trends
📦 Installation & Setup
# Clone the repository
git clone https://github.com/your-username/samatva-ai.git

# Navigate to project folder
cd samatva-ai

# Install dependencies (backend)
pip install -r requirements.txt

# Run backend
uvicorn main:app --reload

# Run frontend
npm install
npm start
